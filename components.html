Security

  • Single entry point

    You can define a single entry point for a group of applications. You can use domain in multi-tenant SaaS context.

  • Password management

    Everything you need to automate this process using email instructions. No more clear text password in email !!!

  • Log in as another user

    For support, you can log in as another user while leaving your own mark in the audits.

  • Declarative security

    Link every APEX pages to an "applicative structure" to use 5 rights level: no access, read-only, modify, create, delete.

  • LDAP integration

    Users can authenticate against external or internal security schemes. You can use an hybrid mode to manage internal staff and public users.

  • Domains

    You can easily build multi-tenant SaaS using domains. Depending on the settings, users in the same user-group but in different domains could have access or not to the same functionality.
